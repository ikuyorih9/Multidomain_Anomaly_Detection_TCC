# DamicorePy

Este diretório contém uma cópia da implementação da DAMICORE, a DamicorePy.

> Acesse o conteúdo original de Kevin Polo: <a>https://gitlab.uspdigital.usp.br/kevin.polo/damicore-python3</a>

As modificações focaram em dois pontos principais:
- `damicore-python3/main.py`: aplicação do método TRICORDER de detecção de defeitos a partir do carregamento incremental de perfis anômalos;
- `damicore-python3/damicore/compressor.py`: inserção dos compressores com e sem perdas;
