# Utils

Este diretório contém scripts gerais, utilizados nos scripts mais específicos para gerar gráficos de barras, mapas de calor, formação de nomes etc.

- `barplots.py`: responsável por gerar os gráficos de barras a partir de um dataframe;
- `boxplots.py`: responsável por gerar boxplots a partir de um dataframe;
- `format.py`: responsável por formatar os dataframes, em relação aos nomes ou ao seu conteúdo;
- `XYPlot.py`: responsável por gerar gráficos (x,y).
